Iniciar una app

Los pasos para iniciar una app con React son los siguientes:

  1. Crear el proyecto con create-react-app nombre_de_la_app.
  2. Cambiar a la carpeta del proyecto con cd nombre_de_la_app.
  3. Iniciar el servidor con npm start (se abre en el navegador automáticamente).
  4. Si no se abre, acceder al servidor escribiendo localhost:3000 en la barra de direcciones del navegador.
